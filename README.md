This is a web application where you can generate you own QR codes and Scan any codes with text and hyperlink for the decrypted content.

üî≠ Features :

1) Generate own QR codes ‚õìÔ∏è
2) scan any type of QR codes and decrypt them üì≤
3) upload and decrypt codes from your local system storageüì±
4) get both text and hyperlink for the decrypted content üñáÔ∏è
